## 🌟 نظرة عامة

منصة التحليل القانوني الذكي هي تطبيق ويب متقدم مصمم خصيصاً للنظام القانوني الفلسطيني. يوفر التطبيق تحليلاً قانونياً شاملاً عبر 13 مرحلة متسلسلة، بالإضافة إلى مساعد قانوني ذكي يعمل بتقنية الذكاء الاصطناعي.

## ✨ الميزات الرئيسية

### 🔍 **التحليل القانوني المتقدم**
- **13 مرحلة تحليلية**: من تحديد المشكلة القانونية إلى العريضة النهائية
- **تحليل تراكمي**: كل مرحلة تبني على السابقة
- **تخصص فلسطيني**: محدود بالقوانين الفلسطينية النافذة
- **مراجع قانونية**: إشارة للمواد والقوانين ذات الصلة

### 🤖 **المساعد القانوني الذكي**
- **محادثة تفاعلية**: استشارة قانونية فورية
- **سياق متخصص**: يفهم المراحل والتحليلات السابقة
- **أمان المحتوى**: يرفض الخروج عن النطاق الفلسطيني
- **اقتراحات ذكية**: خطوات تالية ونصائح عملية

### 📊 **التحليلات والإحصائيات**
- **إحصائيات مفصلة**: تحليل أداء القضايا
- **رسوم بيانية**: تمثيل مرئي للبيانات
- **تتبع التقدم**: معدل إكمال المراحل
- **تحليل الاتجاهات**: أنماط الاستخدام

### 🔐 **نظام المصادقة**
- **حسابات متعددة**: إدارة للمحامين والموظفين
- **أمان متقدم**: تشفير البيانات وحماية الخصوصية
- **إدارة المكاتب**: ربط الموظفين بالمكاتب القانونية
- **جلسات آمنة**: إدارة الجلسات مع انتهاء الصلاحية

### 📱 **تطبيق ويب تقدمي (PWA)**
- **تثبيت كتطبيق**: يعمل كتطبيق أصلي على الأجهزة
- **عمل بدون إنترنت**: للبيانات المحفوظة محلياً
- **تحديثات تلقائية**: Service Worker للتحديثات
- **أداء محسن**: تحميل سريع وتجربة سلسة

## 🚀 التثبيت والتشغيل

### المتطلبات الأساسية
- **Node.js**: الإصدار 18 أو أحدث
- **npm**: الإصدار 8 أو أحدث
- **مفتاح Google Gemini API**

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/your-username/legal_analysis-nextjs-pro.git
cd legal_analysis-nextjs-pro
```

2. **تثبيت التبعيات**
```bash
npm run setup
# أو
cd frontend/pages && npm install
```

3. **إعداد متغيرات البيئة**
```bash
# إنشاء ملف .env.local في frontend/pages/
GOOGLE_API_KEY=your_gemini_api_key_here
NEXT_PUBLIC_ENCRYPTION_SECRET=your_secret_key_here
JWT_SECRET=your_jwt_secret_here
BLOB_READ_WRITE_TOKEN=your_vercel_blob_token_here
```

4. **تشغيل التطوير**
```bash
npm run dev
# أو
cd frontend/pages && npm run dev
```

5. **فتح التطبيق**
افتح [http://localhost:3000](http://localhost:3000) في المتصفح

## 🔧 الإعداد المتقدم

### الحصول على مفتاح Gemini API
1. اذهب إلى [Google AI Studio](https://makersuite.google.com/app/apikey)
2. أنشئ مفتاح API جديد
3. أضفه إلى متغيرات البيئة

### إعداد Vercel Blob (للمصادقة)
1. أنشئ حساب على [Vercel](https://vercel.com)
2. أنشئ Blob Store جديد
3. احصل على BLOB_READ_WRITE_TOKEN
4. أضفه إلى متغيرات البيئة

## 📖 دليل الاستخدام

1. أدخل تفاصيل القضية في الصفحة الرئيسية

2. اطرح أسئلتك القانونية

## 🔐 الأمان والخصوصية

- **تشفير البيانات**: جميع البيانات الحساسة مشفرة
